What is Zedi Token?

The ZEDI Token is a cryptocurrency based on the Ethereum blockchain. ZEDI is an open source, global payment network that is fully decentralized without any central authorities. ZEDI is created to become the cryptocurrency used to pay for transportation and accommodation booking websites.

FAQ

Which wallet should I use to store ZEDI tokens?
ZEDI is an ERC20 token issued on Ethereum blockchain. Token holders can easily store and manage their ZEDI tokens using existing Ethereum clients including official Ethereum wallets like MyEtherWallet, Metamask, Parity, Mist or Ledger (hardware wallet).
Can I participate in the crowdsale without creating an Ethereum address?
No, because ZEDI is a token issued on Ethereum blockchain, so you need to have an Ethereum address to receive and store your ZEDI tokens. You can use any Ethereum wallet: MyEtherWallet, Metamask, Parity, Mist or Ledger.
